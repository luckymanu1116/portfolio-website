# Manoj Guttikonda - Portfolio Website

A modern, responsive portfolio website built with HTML5, CSS3, and JavaScript. 

## Features
- **Neon/Tech Aesthetic**: Dark mode by default with glowing accents.
- **Interactive Timeline**: Experience section with scroll-triggered animations.
- **Dynamic Hero**: Glitch effect and typewriter animation.
- **Responsive Design**: Works on mobile and desktop.

## How to Run

Since this project uses vanilla web technologies, no installation is required.

1.  Simply open the `index.html` file in your web browser.
2.  (Optional) For a better development experience, you can use a Live Server extension in VS Code.

## Structure
- `index.html`: Main HTML structure.
- `style.css`: All styling, animations, and responsive rules.
- `script.js`: Logic for animations, theme toggle, and interactions.
